тЬЕ 1. for, while, do-while Loops in C
ЁЯФБ For Loop
Syntax:

c
Copy
Edit
for(initialization; condition; increment/decrement) {
    // code block
}
Example:

c
Copy
Edit
for(int i = 1; i <= 5; i++) {
    printf("%d ", i);
}
Explanation:

Start from i = 1

Check condition i <= 5

Print i

Increase i by 1, and repeat until condition fails.

ЁЯФБ While Loop
Syntax:

c
Copy
Edit
initialization;
while(condition) {
    // code block
    increment/decrement;
}
Example:

c
Copy
Edit
int i = 1;
while(i <= 5) {
    printf("%d ", i);
    i++;
}
ЁЯФБ Do-While Loop
Syntax:

c
Copy
Edit
initialization;
do {
    // code block
    increment/decrement;
} while(condition);
Example:

c
Copy
Edit
int i = 1;
do {
    printf("%d ", i);
    i++;
} while(i <= 5);
ЁЯЯв Key Difference:
do-while runs at least once, even if the condition is false initially.
